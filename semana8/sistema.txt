El comando uname -a muestra información general del sistema operativo, como el nombre del sistema, la versión del kernel y la arquitectura del equipo.

El comando whoami muestra el nombre del usuario que está utilizando el sistema.

Conocer esta información es útil porque permite identificar el entorno donde se administra un servidor, verificar el usuario activo y facilitar tareas de mantenimiento, configuración y seguridad.
