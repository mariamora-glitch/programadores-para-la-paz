considero que es importante que los proyectos de software mantengan un registro de cambios usando herramientas como git porque permite llevar un control organizado de todas las modificaciones realizadas. gracias a git, los desarrolladores pueden identificar quién hizo cada cambio y cuándo fue realizado. además, facilita corregir errores al poder volver a versiones anteriores del proyecto. también permite que varias personas trabajen en el mismo proyecto sin perder información ni sobrescribir el trabajo de otros. el uso de git mejora la organizaciónla colaboración y la seguridad del desarrollo. por estas razones, es una herramienta fundamental para gestionar proyectos de software de manera eficiente y profesional.
