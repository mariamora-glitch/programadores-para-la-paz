Al enviar una petición POST a la ruta /registro con el siguiente JSON:

{
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}

El servidor responde con:

{
  "estado": "Datos recibidos",
  "nombre": "Maria",
  "mensaje": "Hola comunidad"
}

Esto confirma que Express recibió correctamente los datos enviados mediante req.body y los devolvió en formato JSON.
